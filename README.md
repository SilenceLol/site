# site
Прием заказов на вывоз вторсырья
Веб-приложение для оформления заказов на вывоз вторсырья. Позволяет выбрать дату вывоза и указать вес различных видов отходов.

📱 Возможности
Выбор даты - 3 кнопки с ближайшими доступными датами (воскресенье недоступно)

Учет продукции - таблица с 8 видами вторсырья

Расчет общего веса - автоматический подсчет суммы всех введенных значений

Валидация - минимальный вес заказа 35 кг

Адаптивный дизайн - корректное отображение на всех устройствах

🔧 Настройка
Подключение к реальной базе данных
В файле script.js замените функцию loadAvailableDates():

javascript
// Вместо имитации
setTimeout(() => {
appState.availableDates = generateNextDates(3);
renderDatesButtons();
}, 800);

// Используйте реальный API
fetch('/api/available-dates')
.then(response => response.json())
.then(dates => {
appState.availableDates = dates.map(d => new Date(d));
renderDatesButtons();
});
Изменение минимального веса
В файле script.js найдите строку:

javascript
submitBtn.disabled = !dateSelected || total < 35;
Измените 35 на нужное значение.
